首先要确保环境一致，防止出现环境不一致导致的报错或无法运行，现在确定的有环境不一致导致的test_model文件无法展示图片

安装方法如下（推荐使用最新版本的Anaconda创建环境）
首先使用终端进入该目录下，然后输入
pip install -r requirements.txt

其中只需要注意Pytorch版本可能需要根据你的硬件更改版本。如果是GPU且是10系或者20系显卡需要安装11开头的版本
如果使用的是CPU请安装CPU的版本

！！！注意 ！！！
所有的脚本文件均使用的是相对路径，不要擅自更改，防止路径不正确导致的无法运行！！！！！

训练模型使用的脚本文件名字是trian.py，这个文件主要是训练预训练模型CNN和resnet50
该文件训练完成后会调用train_and_compare.py脚本生成Accuracy，F1，Loss，Recall，Roc线型图，除此之外还会生成一个training_resuts.csv文件记录每一个Epoch的数据。他们都会被保存在对应模型的文件夹内。结构如下
results-----------------------
	compare--------------
		CNN-----------
		ResNet-50----------------

evaluate_model.py脚本用于评估每个预训练模型的最佳权重，运行后在终端输入compare文件夹，例如compare_1。之后输入对应的模型名称即可运行。它会生成evaluation_metrics.txt文件记录对应模型的评估数据

test_model文件是用来可视化查看模型对肺部x光图片的预测准确率（此处为置信度），操作方法和evaluate_model.py一致，最后会生成一个UI框用于展示随机10张图片。
